# audio-translate-server

Audio translate server

## 1. Installing

To install the package from [PyPI](https://pypi.org/project/audio_translate_server/), follow the instructions below:


```bash
pip install --upgrade audio_translate_server
```

Execute `which audio-translate-server` to see where it was installed, probably in `/home/USERNAME/.local/bin/audio-translate-server`.

### Using

To start, use the command below:

```bash
audio-translate-server
```
