# audio-translate-server

Audio translate server


## Install from PYPI

The homepage in pipy is https://pypi.org/project/audio_translate_server/

```bash
pip install --upgrade audio_translate_server
```

Using:

```bash
audio-translate-server
```

## Install from source
Installing `audio-translate-server` program

```bash
git clone https://github.com/trucomanx/AudioTranslateServer.git
cd AudioTranslateServer
pip install -r requirements.txt
cd src
python -m build
pip install dist/audio_translate_server-*.tar.gz
```
Using:

```bash
audio-translate-server
```

## Uninstall

```bash
pip uninstall audio_translate_server
```
